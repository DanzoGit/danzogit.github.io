# Обновление 3.0 B259 EXP

![](./b259.png)

Привет, Выжившие!

Сегодня у нас есть еще одно EXP обновление для всех вас!

Вот как вы можете принять участие:

Нажмите правой кнопкой мыши на игру в Steam

Нажмите "Свойства"

Перейдите на вкладку "Бета-версии"

В выпадающем меню будет доступна latest_experimental <— вам нужно именно ЭТО

Выберите ее и дождитесь скачивания игры

Мы также создали новую ветку под названием: latest_experimental_fallback для всех, кто все еще загружает b252 вместо latest_experimental

Вот что изменилось со времен b257:

**Добавлено**

- Вывод настроек Песочницы (Sandbox) в лог при запуске игры и предоставление консольной команды для просмотра списка опций (getsandboxoptions / gso)
- WebAPI "sandboxsettings", позволяющее просматривать текущие настройки и пользовательские коды, включая все детали настроек песочницы, предоставляемые интерфейсом
- Настройка Песочницы для скорости бега в приседе (Crouch Run Speed)
- Привязка system_time в XUi, возвращающая текущее локальное время и дату (DateTime), чтобы можно было отображать реальное время в интерфейсе
- Методы XUi для регистрации обработчиков атрибутов/привязок XML для существующих контроллеров и представлений

**Изменено**

- Переработан мод "Закаленное лезвие" (Tempered Blade), чтобы иметь те же значения, что и другие моды на урон по блокам. Значения 10-60% уменьшены до 5-30%.
- Переработан мод "Эргономичная рукоятка" (Ergonomic Grip), чтобы предотвратить проблемы с суммированием эффектов с другими перками и предметами. Значения 10-60% уменьшены до 5-30%.
- Переработан мод "Укрепляющая рукоятка" (Fortifying Grip) - добавлена логика, предотвращающая его работу без еды или воды
- Вытягивающие удары (Siphoning Strikes) и Драки в баре, Том 5 (Bar Brawling Vol 5) больше не будут работать без еды или воды
- Обновлены теги на боеприпасах для поддержки перка Удачливый мародер, Том 2 (Lucky Looter Vol 2) и настроек Песочницы
- House_construction_02: место появления спящего медведя заменено на кабана для балансировки сложности
- Обновлен лут в industrial_strip_03, добавлено больше автомобильного лута

**Исправлено**

- Невозможность использовать предметы из инвентаря после того, как предыдущая игровая сессия была прервана
- Настройка Песочницы "Глобальный этап лута" (Global Loot Stage) не влияет на этап лута
- При настройке Песочницы "Скорость приседания" (Crouch Speed) на 300%, она быстрее, чем "Скорость ходьбы" (Walk Speed) на 300%
- Настройка Песочницы "Выпадение при смерти: Удалить всё" (Drop on Death Delete all) не работает
- Настройка Песочницы "Глобальный этап игры" (Global Game Stage) на значение "Низкий" (Low) устанавливала начальный этап игры на ноль
- Кузница позволяет использовать бесплатный рецепт, когда выбран тип плавильни (smelter) в настройках Песочницы
- Невозможность подобрать предметы, размещенные пользователем, у заявленного (claimed) торговца
- Минимальная скорость неправильно использовала настройки модификатора скорости в Песочнице
- Материалы для крафта ограничивали производство предметов, когда стоимость крафта была установлена на "Нет" (None)
- Ввод крафта (Crafting Input) не работал для значения "Нет" (None)
- Неправильный тег на Решетке для гриля (Cooking Grill), позволяющий ей влиять на время приготовления при удержании дополнительной решетки
- Значения цвета в XUi, выходящие за пределы допустимого диапазона 0-255, не ограничивались должным образом, а усекались
- Турель повреждает пространство блока для других устройств
- Восстановлен отсутствующий box collider на bat_Prefab
