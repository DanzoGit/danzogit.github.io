# Improved Building and Blueprints

Устанавливайте целые 3D-массивы блоков сразу, а не по одному. Зажмите `Shift` и кликните правой кнопкой мыши по двум углам, задайте размер области стрелками и нажмите `Enter`. Мод включает в себя полноценную систему создания и размещения чертежей (блюпринтов), пандусы (склоны), инструмент удаления для режима бога, плоскость симметрии и предпросмотр, учитывающий границы приватных территорий. Работает в одиночном режиме и на выделенных серверах.

Устали ставить блоки по одному? Мод **Improved Building** добавляет в 7 Days to Die механику строительства в духе *Empyrion* или *Space Engineers*. Возьмите в руки блок, отметьте два угла, измените размер с помощью клавиш со стрелками и возведите целую стену, пол, колонну или объемную 3D-фигуру за раз, а затем сохраняйте свои постройки в виде многоразовых чертежей.

---

### Основные механики

* **Блочное строительство (Box Build):** Это главная функция мода. Возьмите любой блок, зажмите `Shift + Правая кнопка мыши (ПКМ)`, чтобы начать создание области, затем кликните `ПКМ` на противоположный угол.
* **Изменение размера:** Используйте клавиши со **стрелками**, чтобы изменить площадь основания (она зависит от положения камеры: нажатая стрелка сдвигает границу с соответствующей стороны экрана при любом ракурсе обзора). Удерживайте `Shift` со стрелками, чтобы уменьшить размер стороны.
* **Высота:** `PgUp` и `PgDn` поднимают и опускают выделенную область, при этом `PgDn` опускает её до уровня земли. Зажатый `Shift` вместе с `PgUp` или `PgDn` изменяет высоту сразу на пять блоков.
* **Колонны:** Дважды кликнув по одному месту и нажав `PgUp`, можно быстро построить вертикальную колонну.
* **Режим пандуса:** Клавиша `R` включает режим пандуса, заполняя область пологим склоном вместо сплошных блоков. Повторное нажатие циклично меняет направление подъема: Север, Восток, Юг, Запад, а затем возвращает сплошной блок.
* **Подтверждение / Отмена:** Нажмите `Enter`, чтобы построить, или `End`, чтобы отменить действие.

> **Примечание:** На каждой грани области отображаются зеленые иконки клавиш, показывающие, какая стрелка изменяет её размер. При удержании `Shift` иконки поворачиваются внутрь.

### Чертежи (Blueprints)

Чертежи позволяют сохранять и повторно использовать ваши постройки.

1. Нажмите `Tab`, затем `Insert`, чтобы открыть режим чертежей.
2. **Для захвата:** Отметьте два угла с помощью `ПКМ`, задайте высоту клавишами `PgUp` / `PgDn` и нажмите `Enter` для сохранения.
3. **Для размещения:** Перемещайте проекцию сохраненного чертежа с помощью стрелок, поднимайте или опускайте через `PgUp` / `PgDn`, вращайте клавишей `R` и нажмите `Enter` для завершения постройки.
4. **Редактирование:** Вы также можете корректировать размещение, удаляя отдельные ячейки с помощью `Shift + ПКМ`, после чего нажать `Enter`, чтобы сохранить и выйти. В окнах подтверждения всегда используется `Enter` для ответа «Да» и `End` для ответа «Нет».

### Плоскость симметрии

Плоскость симметрии зеркально отражает размещаемые блоки для быстрого симметричного строительства. Активируйте её через меню (`Tab`, затем `Insert`) и кликните `ПКМ`, чтобы установить: теперь каждый поставленный вами блок будет отзеркален по выбранной оси.

### Режим удаления (только для режима бога / god mode)

Позволяет быстро расчищать территорию. Выделив область, нажмите `Delete` для активации режима.

* Нажатие `Enter` очистит только ландшафт (землю) внутри области.
* Нажатие `Shift + Enter` удалит вообще всё, включая построенные игроком блоки. Отличный инструмент для масштабных раскопок и сноса зданий.

### Защита территорий (Land-claim)

**Improved Building** учитывает границы приватных территорий. Он позволяет строить только внутри вашего собственного привата или привата союзников; за их пределами сетка предпросмотра подсвечивается красным. Защита территорий строго соблюдается даже в режиме бога, поэтому мод абсолютно безопасен для использования на многопользовательских серверах.

### Встроенная помощь

Встроенная панель помощи со списком основных горячих клавиш отображается при появлении в игре. Нажмите `End`, чтобы скрыть её; панель можно в любой момент снова вызвать через меню по клавише `Insert`.

### Совместимость и Установка

Мод полностью протестирован на выделенных серверах и автоматически определяет свою роль (одиночная игра, клиент или выделенный сервер). Для многопользовательской игры мод необходимо установить **и на сервере, и на подключающихся клиентах**.

* **Совместимость:** 7 Days to Die v2.6, одиночная игра и выделенные серверы. Античит EAC должен быть отключен, так как это кодовый мод на базе библиотеки Harmony.
* **Установка:** Скачайте и распакуйте архив, затем поместите папку `ImprovedBuilding` в вашу папку `7 Days To Die/Mods/` (создайте папку *Mods*, если её не существует). На выделенных серверах скопируйте ту же папку в директорию *Mods* вашего сервера. Запускайте игру с отключенным EAC.

### КРАТКОЕ РУКОВОДСТВО (горячие клавиши)

| Действие | Комбинация клавиш |
| --- | --- |
| **Начать область** | `Shift` + `Правая кнопка мыши (ПКМ)` |
| **Отметить противоположный угол** | `ПКМ` |
| **Изменить площадь основания** | `Стрелки` |
| **Уменьшить сторону** | `Shift` + `Стрелки` |
| **Поднять / опустить область** | `PgUp` / `PgDn` |
| **Высота +/- 5 блоков** | `Shift` + `PgUp` / `PgDn` |
| **Построить колонну** | Двойной клик на одно место + `PgUp` |
| **Режим пандуса (смена направления)** | `R` |
| **Построить / Отменить** | `Enter` / `End` |
| **Режим чертежей** | `Tab` -> `Insert` |
| **Повернуть чертеж** | `R` |
| **Режим удаления (только god mode)** | `Delete`, затем `Enter` |
| **Да / Нет в окнах подтверждения** | `Enter` / `End` |
